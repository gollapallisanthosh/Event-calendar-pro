# 📅 Event Calendar Pro

A premium, fully interactive Event Calendar web application built using HTML, CSS, and JavaScript.  
This project provides a modern UI and powerful event management features similar to Google Calendar.

## 📌 Overview

Event Calendar Pro is a dynamic web application that allows users to manage their daily schedules efficiently. It includes advanced features like drag-and-drop scheduling, dark mode, and persistent data storage using localStorage.

This project demonstrates strong frontend development skills, including DOM manipulation, event handling, and UI/UX design.

---

## 🚀 Features

### 📅 Calendar Functionality
- Dynamic monthly calendar generation
- Navigate between months and years
- Highlights current day and selected day
- Displays previous and next month dates for full grid layout

### 📝 Event Management (CRUD)
- ➕ Add events with time range
- ✏️ Edit existing events
- 🗑️ Delete events
- 📌 Highlight days with events

### 🔄 Drag & Drop (Advanced)
- Move events between dates using drag-and-drop
- Real-time UI update after drop
- Uses HTML5 Drag & Drop API

### 🌙 User Experience
- Dark mode toggle
- Smooth animations and transitions
- Interactive hover effects
- Clean and modern UI (Glassmorphism style)

### 💾 Data Persistence
- Stores all events using localStorage
- Data remains even after page refresh

## 🧠 How It Works

### 1. Calendar Rendering
- JavaScript dynamically generates calendar days
- Calculates:
  - First day of month
  - Last day of month
  - Previous/next month filler days

### 2. Event Storage
- Events are stored as objects:

- Saved in browser using localStorage

### 3. Event Handling
- Events are filtered based on selected date
- DOM updates dynamically to display events

### 4. Drag & Drop Logic
- `dragstart` → store event ID  
- `drop` → update event day  
- UI refresh → reflects changes instantly
